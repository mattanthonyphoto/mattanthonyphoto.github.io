Home/Projects/Residential/Summer Lane

Set on beautiful Green Lake and surrounded by panoramic views of the Whistler mountains, this project is sure to be a solace retreat for years to come. Features an expansive kitchen, library, wine room, and loft area. Design combines transitional elements with greige teak shaker cabinets and flat-panel white millwork, hickory hardwood on main level, polished concrete on lower level. Approximately 4000 sq ft with 3 bedrooms, 4 bathrooms.
